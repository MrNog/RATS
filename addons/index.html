📥 RATS — Addons

Every download is the latest version straight from the source. A gold NEW badge means there's an update you haven't downloaded yet — it clears when you hit Download.

Install: download the .zip → extract → drop the addon folder into World of Warcraft\Interface\AddOns\ → /reload. If the extracted folder ends in -main / -master, rename it to remove that suffix.

📥 Mandatory addons — every raider

✨ WeakAura tier packs — important for raids

Import in-game: open a pack → copy the code → /wa → Import → paste.

🧩 Visual patch — optional (non-HD)

Drop patch-y.mpq in World of Warcraft\Data\ (NOT AddOns) → restart. Works on the normal 3.3.5a client.

🐀 Cool extras — optional

Not required, but handy or just fun. Install only if you want them.
