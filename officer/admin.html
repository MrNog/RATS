🛠 RATS — Admin Console

Manage the guild key, the lock, and the roster. This console is for officers/GM only.

🔑 Guild key & lock

The key officers type to unlock the tools. Set it here and arm the lock — this works with no roster needed.
Commit the downloaded gate.json to the repo → the index lock turns on. Share the key in the officer channel.

📦 Roster

Paste your in-game guild export, then export it encrypted with the guild key above.
Saves instantly to all officers when Firebase is on; otherwise downloads roster.json to commit via Fork. The Guild Roster + comp autocomplete read it.

💾 Backup

Download the current encrypted roster.json + history.json and commit them via Fork. Each commit is a frozen, unchangeable snapshot. If the live data is ever wiped, the site restores from these automatically.
Tip: do this after big roster/raid changes. Your browser may ask to allow multiple downloads — say yes.

📣 Discord webhooks

One per channel/purpose — give each a name so tools know which is which (e.g. Raid Roster, Lore / Stories). Stored only in your browser — never committed.
The comp tool posts to a hook named like roster/raid; the Lore page uses one named like lore/story (or you pick at post time).

🛠 Admin password

Locks this console (separate from the guild key — only you should know it).
Commit the downloaded admin.json → this console requires that password.
