🔧 RATS — Admin Console

Manage the guild key, the lock, and webhooks. Officers / GM only.

🔑 Guild key & lock

The key officers type to unlock the tools. Arm the lock — works with no roster needed.

Commit the downloaded gate.json → the index lock turns on. Share the key in the officer channel.

💾 Backup

Download the current encrypted roster.json + history.json and commit via Fork. Each commit is an unchangeable snapshot — restores automatically if live data is wiped.

Tip: do this after big roster/raid changes. Your browser may ask to allow multiple downloads — say yes.

📣 Discord webhooks

Names are fixed — tools find the right hook by name. Only paste/edit the URL. Leave blank to disable. Stored only in your browser — never committed.

🔧 Admin password

Locks this console (separate from the guild key — only you should know it).

Commit the downloaded admin.json → this console requires that password.
