📊 How attendance works — and why (click)

Attendance is not a punishment — it shows who's the most reliable, so raid spots, loot priority and 💀 Fang promotions are fair. A miss is just data.

One lockout = one obligation, however many nights it took. A run split over 2–3 nights (couldn't full-clear in one sitting) still counts once. Your % = how much of the run you actually did — came to 1 of 2 nights = 50%, every night = 100%. This is what feeds loot council. Runs = how many lockouts you showed up to.

Two kinds of miss (both shown as pills, so you can see who's flaky):

  • 🔴 signup and ghosted — you signed up and never showed. Marked by an officer.
  • 🟡 didnt show 2nd day — you came the first night then didn't return for a later night of the same run. Detected automatically. (Coming late to help finish is not this — that just counts as the partial you did.)

🏖️ On vacation = excused. New members are protected — raids before their join date don't count. An alt counts for its main.

Which raids count:

  • 🔴 25-man — Mandatory. Counts for everyone from their join date.
  • 💀 10-man — Fang run. Counts for Warchief's Fangs only (and anyone who actually played it).
  • ⚪ Optional — log-only. Alt 25 runs, casual/pug 10-mans: saved as history only, counts for nobody.

Flip any run with the Optional switch on its card in the log — it saves instantly.
