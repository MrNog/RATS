Guild Roster →
Import the guild roster: browse members by rank, search, see mains & alts. Powers name autocomplete in the comp tool.
Raid Comp Viewer →
Paste a Raid-Helper composition, render it with spec icons + no-shows, and post it to Discord as a clean embed.

Saved raids over time + per-raider attendance %. Alts count for their main; each raid day counts separately. Mandatory-day tracking.

See who's away and add a raider's vacation dates — excused absences don't hurt their attendance.

Write tales about the raid — markdown + images — and post them to Discord. For the storytellers of the Rats. 🐀
Officer Files →
Shared files for officers — sheets, exports, docs. Hosted here, one-click download. No external drive.
